# 02 Networking

Segundo nível da trilha. O foco aqui é comunicação entre processos, concorrência e servidores simples.

## Projetos

### [chat-server-client](./chat-server-client/README.md)

Exemplo com cliente e servidor TCP para praticar:

- `TcpListener` e `TcpStream`
- threads
- canais (`mpsc`)
- comunicação em rede

### [web-server](./web-server/README.md)

Servidor web simples para praticar:

- sockets TCP
- tratamento básico de requisições HTTP
- organização em biblioteca e binário
- thread pool
